# Basic Types
* auto
  * default: 0
* bool
  * default: false
* int
  * default: 0
* double
  * default: 0.0
* string
  * default: ""

# Array
* Define
  * `int a[]` or `int a[10]`
* Access Element
  * First Element: `a[0] = 1`
  * Last Element: `a[-1] = 1`
  * All Elements: `a[] = 1`
  * Range Elements: `a[0..2] = 1`
* Access Length
  * Increase Length: `a++` or `a += 2`
  * Decrease Length: `a--` or `a -= 2`
  * Define Length: `a = 10`
  * Get Length: `a`

# Kommentare
* `//` or `#` Single Line
* `/* */` Multi Line
